# Architecture Decision Records (ADR)

This directory contains the architecture decision records for the Distributed Rate Limiter project.

## List of ADRs

- [ADR-001: Token Bucket Algorithm](001-token-bucket-algorithm.md) - Rationale for choosing Token Bucket as the default algorithm.
- [ADR-002: Redis Distributed State](002-redis-distributed-state.md) - Design for distributed state management using Redis.
- [ADR-003: Fixed Window Algorithm](003-fixed-window-algorithm.md) - Implementation of the memory-efficient Fixed Window algorithm.
- [ADR-004: Leaky Bucket Algorithm](004-leaky-bucket-algorithm.md) - Implementation of the traffic-shaping Leaky Bucket algorithm.
- [ADR-006: Adaptive Rate Limiting](006-adaptive-rate-limiting.md) - Design for ML-driven adaptive rate limiting.
